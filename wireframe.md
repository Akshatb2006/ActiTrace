# Overview

This document describes the low-fidelity wireframes for the ActiTrace application.

ActiTrace is an ML-powered Human Activity Recognition system that:
- Accepts wearable sensor data
- Processes it using an XGBoost classification model
- Stores predictions and metadata
- Displays activity timelines, summaries, and analytics

The wireframes focus on:
- Functional clarity
- Backend integration mapping
- User flow visibility
- Error handling
- ML system transparency
